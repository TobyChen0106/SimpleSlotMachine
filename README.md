# Simpel Slot Machine Game
- 🚀 [deploy](https://tobychen0106.github.io/SimpleSlotMachine/)

## Prerequisite
1. node (可至 [官網](https://nodejs.org/en/download/) 下載)

## how to install
1. npm install

## how to build
1. npm run build

## how to run
1. npm start

## Run WS test server
1. node we_server.js (模擬出鈔機的 ws 伺服器)

## how to run test (on localhost:3000)
1. npm startlocal
